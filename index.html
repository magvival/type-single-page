What is Roboto Flex?

Roboto Flex is a Variable font branched from the original font Roboto; a neo-grotesque sans serif type family. Created by Christian Robertson who was an in-house designer at Google. which was used in 2011 for the system font for Android products. Roboto Flex was later released almost a decade later in 2022. A variable adaptation of the font was created. The variable font implemented 12 different adjustable axes and 50 languages to choose from. It was created by 6 type designers by the names of David Berlow, Santiago Orozco, Irene Vlachou, Ilya Ruderman, Yury Ostromentsky, and Mikhail Strukov.

Supported Languages:

50 languages: Afrikaans, Albanian, Azerbaijani, Basque, Belarusian, Bosnian, Bulgarian, Catalan, Croatian, Czech, Danish, Dutch, English, Estonian, Faroese, Filipino, Finnish, French, Galician, German, Greek, Hungarian, Icelandic, Indonesian, Irish, Italian, Kazakh, Latvian, Lithuanian, Macedonian, Malay, Mongolian, Norwegian Bokmål, Polish, Portuguese, Romanian, Russian, Serbian, Slovak, Slovenian, Spanish, Swahili, Swedish, Tongan, Turkish, Ukrainian, Uzbek, Vietnamese, Welsh, Zulu.
